    管理領域

    メニュー項目の下、盤面の上に管理領域という部分があります。[設定]メニューなどから 設定できる項目の一部はここからでも設定可能です。

    共通マウス操作

    各パズルは、マウスから数字や黒マス、境界線などを入力することができます。 「数字・記号をパネルで入力する」のチェックボックスがonになっている場合は、 ポップアップの数字をクリックすることでも数字などを入力することができます。 詳細は各パズルの操作説明を参照してください。

    共通キーボード操作

    数字キー
    カーソルの位置に数字を入力する
    矢印キー or Alt+HJKLキー
    カーソルを移動する
    -(マイナス)キー
    問題入力モード時、カーソルの位置に？を入力したり消去する
    DELキー/Spaceキー
    カーソルの位置の数字・文字を消去する
    Shift/Commandキーを押し続ける
    マウスの左右のボタンを反転する
    Ctrl/Commandキー+Zキー
    操作をUndoする
    Ctrl/Commandキー+Yキー
    操作をRedoする
    Zキーを押し続ける
    線や黒マスの色分け機能がある場合、クリックした部分を着色する
    Alt+Cキー
    (エディタのみ)回答/問題入力モードを切り替える
    F2キー
    (エディタのみ)回答入力モードへ移る
    Shift+F2キー
    (エディタのみ)問題入力モードへ移る
    Ctrl+F8キー or Alt+Pキー
    デバッグ用ウィンドウを表示する

    Management-Area

    There is Management-Area between the menus and the board. Some features that is in 'Setting' menu is able to be set here.

    Mouse Usage

    You are able to input numbers, black cells, borders, etc. by mouse or some pointer devices. If checkbox 'Input numbers by panel' is checked, you are able to input numbers or marks by displayed panel. Details are referred each puzzle help menu.

    Keyboard Usage

    Number keys
    Input number at the cursor.
    Arrow keys or Alt + HJKL key
    Move the cursor.
    -(minus) key
    Input or Erase question marks at the cursor (in edit mode).
    DEL/Space key
    Clear at the cursor.
    Pressing Shift/Command key
    Toggle Mouse Left/Right button.
    Ctrl/Command + Z key
    Undo
    Ctrl/Command + Y key
    Redo
    Pressing Z key
    Emphasis connected line or black cells if the feature enabled.
    Alt + C key
    (Editor only) Toggle edit or play mode.
    F2 key
    (Editor only) To edit mode.
    Shift + F2 key
    (Editor only) To play mode.
    Ctrl + F8 key or Alt + P key
    Display debug popup.

    [ファイル]メニュー

    新規作成
    指定したサイズの盤面を新規に作成します。現在の盤面は消去されます。
    URL入力
    URLを入力して盤面を表示します。ぱずぷれ(v3/アプレット)、カンペン、 へやわけアプレットに対応しています。現在の盤面は消去されます。
    URL出力
    現在の盤面をWebで公開する為のURLを生成します。ぱずぷれv3、カンペン、 へやわけアプレットのURLを出力することができます。
    ファイルを開く
    ぱずぷれv3、pencilboxで出力されたファイルを開くことができます。 現在の盤面は消去されます。
    ファイル保存
    ぱずぷれv3形式、pencilbox形式のファイルを保存できます。
    画像を保存
    現在の盤面の画像を別タブで開くか、画像としてpng形式もしくはSVG形式でダウンロードできます。 カーソルは表示されません。
    ※Internet Explorer9.0+, Firefox3.5+, Google Chrome 2.0+, Safari 4.0+, Opera 10.50+のみ使用できます

    [編集]メニュー

    履歴
    UndoしたりRedoしたりできます。
    盤面
    正答判定や回答の消去を行います。
    [盤面] - チェック
    盤面の正答判定ができます。
    [盤面] - 回答消去
    回答情報を全て消去します。
    [盤面] - 補助記号消去
    補助記号を全て消去します。
    盤面の調整
    盤面のサイズを調整します。上下左右方向のサイズを1ずつ増やしたり減らしたりできます。
    回転・反転
    盤面の90°回転あるいは上下・左右の反転を行うことができます。
    盤面の複製
    現在の盤面を複製します。履歴も複製されます。
    ※Internet Explorer8.0+, Firefox2.0+, Google Chrome 5.0+, Safari 4.0+, Opera 10.50+のみ使用できます

    [File] Menu

    New Board
    Create new board whose size is set. Current board is wiped out.
    Input URL
    Import URL and display it. URL of PUZ-PRE (v3/applet), Kanpen and Heyawake-Applet is enabled. Current board is wiped out.
    Output URL
    Export URL that is used for releasing on Web. URL of PUZPRE v3, Kanpen and Heyawake-Applet is enabled.
    Open the file
    Open the file that is saved from PUZPRE v3 and pencilbox. Current board is wiped out.
    Save the file as ...
    Save the PUZPRE v3 or pencilbox file.
    Save the Image
    Open another tab and output the image of current board there or download it as png or svg format. The cursor is hidden.
    - This feature is enabled with Internet Explorer9.0+, Firefox3.5+, Google Chrome 2.0+, Safari 4.0+ and Opera 10.50+.

    [Edit] menu

    History
    Undo or Redo menu.
    Board
    Check or erase the answer.
    [Board] - Check
    Check the answer.
    [Board] - Erase answer
    Erase the answer.
    [Board] - Erase auxiliary marks
    Erase auxiliary marks.
    Adjust the board
    Adjust the board size. Increase or decrease one step for four direction.
    Turn/Filp the board
    Turn quarter or X/Y-filp the board.
    Duplicate the board
    Duplicate current board with history.
    - This feature is enabled with Internet Explorer8.0+, Firefox2.0+, Google Chrome 5.0+, Safari 4.0+ and Opera 10.50+.

    [表示]メニュー

    表示サイズ
    盤面の大きさを指定します
    [表示サイズ] - サイズ指定
    盤面のサイズを指定します。セルのサイズをピクセル単位で入力します。
    [表示サイズ] - 表示倍率
    「サイズ指定」で指定したサイズの拡大率を指定します。
    フォント
    文字をゴシック体で描画するか、明朝体で描画するか指定します。
    線の色分け
    線の自動色分け機能がついているパズルで、色分け機能の有効/無効を切り替えます。
    カーソルの表示
    カーソルの表示/非表示を切り替えます。
    自動横幅調節
    ウィンドウに入るように盤面のサイズを調節する機能の有効/無効を切り替えます。
    盤面の再描画
    盤面全体を再描画します。数字がずれた時などに使用してください。
    管理領域を隠す/管理領域の表示
    メニュー下のチェックボックスがある領域の表示/非表示を切り替えます。

    [設定]メニュー

    モード
    (エディタ)問題入力モード、回答入力モードを切り替えます。
    入力方法
    黒マスを入力するパズル等で、左右ボタンで黒マス/補助記号別々に入力するか、 なし->黒マス->白マス->なしでループさせるか設定できます。シャカシャカはパズル個別の 入力設定を参照してください。
    繋がりチェック
    黒マス、線を入力するパズルでひと繋がりになった部分を表示するモードにします。 Zキーを押し続けたのと同じ状態になります。
    正答自動判定
    正解判定を自動で行い、正解になったら通知するようにします。
    マウス左右反転
    マウスの左右ボタンを反転します。Shiftキーや、MacでCommandキーを押した時と 同じ効果になります。
    パネル入力
    マウスから数字・記号を入力する為のポップアップを表示します。
    盤面にキー入力
    盤面をキー入力の対象に設定します。
    言語
    言語を切り替えます。
    (その他)
    そのほか、パズルによって様々な項目が追加されています。

    [その他]メニュー

    ぱずぷれv3について
    creditやバージョンを表示します。
    操作説明
    このページを開きます。

    [Display] menu

    Cell Size
    Set the board size by cell size.
    [Cell Size] - Cell Size
    Set the pixel size of the cells.
    [Cell Size] - Display size
    Set the zoom ratio of the board.
    Font
    Set the board font either sans-serif or serif.
    Encolor lines
    If the puzzle has the feature, set the encolor line feature enabled.
    Display cursor
    Toggles to display the cursor on the board.
    Auto Size Fix
    Set the feature that reduces the size in order to squeeze up the whole board.
    Repaint the board
    Repaint the whole board. It is used if the number is not in the proper position.
    Show/Hide management area
    Show/Hide the Management-Area.

    [Setting] menu

    Mode
    (EDITOR) Change the mode whether edit ot play mode.
    Use
    Change the mouse usage whether left/right click inputs each black or white cells or loop empty->black cell->white cell->empty if the puzzle need black cells. Shakashaka is special setting, so please refer the 'How to Input this puzzle'.
    Continueous Check
    Set the mode that emphasis the continueous line or black cells. It is the same state with pressing Z key.
    Automatically check
    Check the asnwer automatically and notice if the board is complete.
    Flip mouse button
    Set mouse left/right button inversed. It is the same state with perssing Shift key or Command key on Mac.
    Display panel
    Display the popup in order to input numbers or marks by mouse directly.
    Key input to Canvas
    Assign the canvas to target key input.
    Language
    Set the PUZPRE v3 language.
    (others)
    Some features are added in some puzzles.
